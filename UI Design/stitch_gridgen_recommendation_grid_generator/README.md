# Note

This design was generated using Google Stitch. It should be treated as an approximation of the intended UI design (i.e. a prototype) rather than as a fixed set of requirements. That being said, this is quite a good prototype and it is very close to the intended UI.

The intended design should be made using modular shadcn UI components. We want a very sleek user experience that feels modern and professional. It should work nicely at any resolution and be naturally adaptive to changes in resolution (i.e. changing the window size should not break it).
